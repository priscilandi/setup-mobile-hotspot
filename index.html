How to Add a Printer in Windows

Objective

To successfully add a network printer or install a new local (USB/Wi-Fi) printer on a Windows computer.

Prerequisites

  • For Network Printer: Network access & server name.
  • For New Printer: Printer with cables or Wi-Fi details.
  • Administrator credentials may be required.

Use this method for office printers already available on a server.

  1. Open File Explorer.
  2. In the address bar, type the server path \\spsua00 and press Enter.
  3. Find your printer in the list and double-click it to install.
  4. If prompted for credentials, contact your IT department for assistance.

1. Connect and Power On the Printer

  • USB Printer: Connect the USB cable from the printer to your computer and turn the printer on. Windows will attempt to set it up automatically.
  • Wi-Fi Printer: Use the printer's display menu to connect it to your Wi-Fi network first. You will need your Wi-Fi name and password.

2. Add the Printer in Windows

  • Go to Settings > Bluetooth & devices > Printers & scanners.
  • Click the Add device button.
  • Wait for your printer to appear in the list, then click Add device next to its name.

3. Install Driver (If Needed)

If the printer doesn't work, you may need to install a driver. The best method is to visit the printer manufacturer's support website (e.g., HP, Canon, Epson), search for your model, and download the recommended software.

4. Print a Test Page

  • In Printers & scanners, click on your printer.
  • Select Printer properties, then click Print Test Page. If it prints, the setup is complete.

Process Complete

Your printer should now be configured and ready for use.
